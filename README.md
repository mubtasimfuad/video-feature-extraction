# Video Feature Extraction Tool

A Python tool for analyzing video files and extracting key visual and temporal features.

## Features
- Shot Cut Detection
- Motion Analysis using Optical Flow
- Text Detection (OCR)

## Requirements
- Python 3.11+
- OpenCV
- NumPy
- pytesseract
